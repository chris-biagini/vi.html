vi.html

A markdown editor with vim keybindings

Modes

Vim has distinct modes for different tasks. The current mode is shown in the status bar.

ModeEnter withPurpose
NormalEscNavigate, delete, copy, paste, and issue commands. This is your home base.
Inserti a o A OType text. Press Esc to return to Normal.
Visualv (char) · V (line)Select text, then act on it (delete, yank, reflow, etc.).
ReplaceROverwrite characters in place.

Text Objects

Use with operators like d, c, y, gq. Prefix with i for “inner” or a for “around”.

ObjectMeaningExample
wWorddiw = delete inner word
WWORD (whitespace-delimited)daW = delete around WORD
sSentencecis = change inner sentence
pParagraphgqap = reflow around paragraph
" ' `Quoted stringci" = change inside quotes
( [ {Bracket pairdi( = delete inside parens

Search

KeyAction

Counts

Most commands accept a numeric prefix: 5j moves down 5 lines, 3dd deletes 3 lines, 2dw deletes 2 words.

Text Reflow

Set :set tw=72 (or your preferred width), then use gq with a motion or text object to hard-wrap text:

KeyAction
gqqReflow current line
gqapReflow current paragraph
gq}Reflow from cursor to next blank line
gqjReflow current line and next
Visual + gqReflow selected lines

Lines are wrapped at word boundaries. Indentation is preserved. This is useful for writing semantic line breaks.

Persistence

  • Content is auto-saved as you type (1-second debounce). Saved content expires after 7 days of inactivity. Use :w to save explicitly and refresh the timer.
  • Settings (:set values) persist indefinitely.
  • Disable auto-save with :nopersist. Re-enable with :persist.
  • Clear saved content immediately with :clear.

Preview & SmartyPants

The Preview tab renders your markdown with full GitHub Flavored Markdown support: tables, task lists, strikethrough, and fenced code blocks.
